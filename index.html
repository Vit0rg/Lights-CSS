LIGHTS

Para essa implementação, encontrei 3 soluções. A primeira(SVG) usa animação vetorial, porém para aplicações responsivas não é viável.

A segunda(span) e terceira(div) forma são bem parecidas entre si, ambas aplicam elementos vazios em HTML, que são mais fáceis de tornar responsivos devido a sua flexibilidade.

Em todos os casos, a animação do arco-íris foi feita com keyframes (em puro CSS3).

For this implementation, I found 3 solutions. The first (SVG) uses vector animation, however for responsive applications it is not feasible.

The second(span) and third(div) shape are very similar to each other, both apply empty HTML elements, which are easier to make responsive because of its flexibility.

In all cases, the rainbow animation was done with keyframes (in pure CSS3).
